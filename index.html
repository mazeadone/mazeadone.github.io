How this works

  1. Scan the Starter QR at the ACM table (or press Start if you're already there).
  2. Solve the puzzle. The answer unlocks a hint for the next location.
  3. Go to that location and scan the posted QR to get the next clue page.
  4. Finish the last clue and come back to the ACM table.

Heads up: this is a static GitHub Pages site (no server). Progress is stored only on your device.
